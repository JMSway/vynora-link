# vynora-link

Damir's links page. Single-file vanilla HTML/CSS/JS.

## Deploy

1. Push this repo to GitHub.
2. Connect to Cloudflare Pages → no build command, output dir = `/`.
3. Set custom domain (e.g. `links.growup-method.pages.dev` or standalone).

## Local preview

```
python3 -m http.server 8000
# open http://localhost:8000
```

## Avatar

Drop a square image at `assets/avatar.jpg`. If missing, the avatar falls back to a "D" monogram automatically.
